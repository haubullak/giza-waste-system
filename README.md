# منظومة المخلفات الصلبة - محافظة الجيزة

## 📋 عن المشروع
نظام متكامل لإدارة تراخيص البناء والمخلفات الصلبة في محافظة الجيزة، يشمل:
- تسجيل طلبات جديدة مع حساب الرسوم تلقائياً
- صلاحيات متعددة للمستخدمين (موظف حي، مدير حي، موظف هيئة، مدير هيئة)
- تحديد موقع العقار على الخريطة
- تقارير Excel و PDF
- استرداد المبلغ التأميني

## 🔧 التقنيات المستخدمة
- HTML5 / CSS3 / JavaScript
- Supabase (قاعدة البيانات)
- Vercel (الاستضافة)
- Leaflet (الخرائط)

## 🚀 طريقة التثبيت والنشر

### 1. إنشاء قاعدة بيانات Supabase
- سجل في supabase.com
- أنشئ مشروع جديد
- نفذ كود SQL الموجود في ملف `supabase-schema.sql`

### 2. تعديل ملفات الاتصال
- افتح ملف `supabase-client.js`
- ضع `SUPABASE_URL` و `SUPABASE_ANON_KEY` الخاصين بمشروعك

### 3. النشر على Vercel
```bash
npm install -g vercel
vercel --prod
